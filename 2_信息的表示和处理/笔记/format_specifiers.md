# 1. Format Specifiers
- "int64_t": PRId64
- "uint64_t": PRIu64
- 指针: %p
- sizeof(): %lu
- int: %d
- unsigned int: %u
- 16进制: %x
- size_t: %lu

# 2. 等价类型
- `unsigned int` <--> `unsigned`: 4 bytes
- `size_t` <--> `unsigned long`: 4 bytes
